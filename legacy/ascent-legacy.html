Ship It

Make it yours, online, forever.

This whole tracker is one self-contained file. Your progress lives in your browser's local storage, so it sticks around between visits once you open it as a real page (not the preview). Here are three ways to get it live — pick the easiest.

💾 Heads up on saving: progress is stored per-browser via localStorage. It persists when you open index.html from your disk or a deployed URL. In sandboxed previews it may reset — that's expected. Use the Export backup button below before switching machines.

Option A — GitHub Pages (free, permanent URL)

Create a repo

On GitHub, make a new public repo, e.g. dsa-ascent.

Add the file

Upload index.html to the repo root (drag & drop on github.com works).

Turn on Pages

Settings → Pages → Source: main branch, /root. Save.

Visit your URL

In ~1 min it's live at username.github.io/dsa-ascent. Bookmark it.

Option B — Netlify / Vercel (drag & drop)

Go to the drop zone

Open app.netlify.com/drop (no account needed to start).

Drag the file

Drop index.html (or its folder) onto the page.

Done

You instantly get a live URL like ascent-xyz.netlify.app.

Option C — Just run it locally

Double-click index.html to open in any browser. Or serve it:

# from the folder containing index.html
python3 -m http.server 8080
# then open http://localhost:8080

Backup & restore your progress

JSON

Export saves everything — checkmarks, dates, notes, catch-up days, contest log — to a file. Import on any device to continue exactly where you left off.
